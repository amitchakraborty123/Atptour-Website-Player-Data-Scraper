
# Atptour Website Player Data Scraper

This script based on property data scrap from https://www.atptour.com/en/rankings/singles


## What you will get:

- All data in in a CSV file. 
- Data points are- Link, Name, Age, Weight, Height, Turned pro, Player Social Media, Country, Birthplace, Plays, Coach, ATP Tour & Grand Slam, Challenger Tour, ITF


## Requirement

- Language: Python
- Libraries: pandas, Datetime, selenium, Bs4, lxml, os


## Documentation

- Install python on your OS. You can install latest version of python.
- Now open your cmd/terminal from the script folder and run this command:
```bash
  pip install -r requirements.txt
```
- Now we can run the script using this command from that folder terminal/cmd:
```bash
python atptour.py
```
- After run, it will ask to input url. Put your targeted url & hit enter. Example URL: https://www.atptour.com/en/rankings/singles?rankDate=2021-11-22&rankRange=0-500
- It will automaticaly go through all player pages. Save the data in a CSV file.
